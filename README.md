# TruckManager

Десктопное приложение для учёта работы спецтехники. Позволяет вести полный контроль над каждой единицей техники: расходы на запчасти, смены водителей, рейсы, зарплата и финансовая отчётность — всё в одном месте. Работает локально на Mac, данные хранятся в SQLite без необходимости интернета или сервера.

## Возможности

- **Рейсы** — учёт поездок по водителям, технике и проектам, автоподстановка тарифа по сетке километраж/ставка
- **Расходы** — учёт затрат по категориям, статусам оплаты, поставщикам и проектам
- **Склад** — остатки запчастей, движение, импорт счетов из PDF
- **Сотрудники** — карточки, смены, расчёт зарплаты
- **Проекты / Клиенты** — детализация по рейсам, расходам, проживанию; экспорт в PDF и Excel
- **Поступления** — учёт входящих платежей от заказчиков
- **Организации** — справочник поставщиков и клиентов, учёт долгов
- **Отчёты** — KPI, сводка по технике и водителям, графики по месяцам, долги
- **Проживание и питание** — расчёт расходов по объектам и сотрудникам
- **Аудит** — история всех изменений в системе
- **Бэкап** — создание и восстановление резервных копий базы данных
